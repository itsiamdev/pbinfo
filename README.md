# PbInfo Solutions

C++ solutions explained step by step for problems from pbinfo.ro, designed for high school students (grades 9–12).

## Features

- Search problems by ID, title, category, or code content
- Filter by difficulty (easy, medium, hard) and category
- C++ code displayed with syntax highlighting
- Modern interface with Tailwind CSS

## Development

```bash
npm install
npm run dev
```

## Structure

- `src/routes/` - Application routes (TanStack Router)
- `src/components/` - Reusable components
- `src/data/problems.ts` - Problem data
